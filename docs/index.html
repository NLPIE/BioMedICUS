What is BioMedICUS?

The Biomedical Information Collection and Understanding System (BioMedICUS) is a system for large-scale text analysis and processing of biomedical and clinical reports.

BioMedICUS is used at the University of Minnesota for the processing of millions of notes, and with a minimal number of dependencies and prerequisites for usage, BioMedICUS is built to promote scalability, performance, and usability.

What does BioMedICUS do?

BioMedICUS is an NLP pipeline which takes text and runs the following processing tasks on it:

  • Sentence Detection
  • Section Detection
  • Tokenization
  • Number Normalization
  • Part of Speech Tagging
  • Word Normalization
  • Acronym Detection
  • UMLS Concept Labeling

In addition, BioMedICUS contains an RTF parser which allows BioMedICUS to run on RTF documents, and also use the structured formatting information during processing.
