typesNotDefaulted

abstract fun typesNotDefaulted(): MutableList<String>

A list of the types where the default action is not taken. If the default action is to delete, these types will be copied to the new view. If the default action is to copy, these types will be deleted.

Return
list of qualified UIMA type names where the default action is not taken
