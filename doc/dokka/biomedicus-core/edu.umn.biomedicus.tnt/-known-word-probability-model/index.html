KnownWordProbabilityModel

open class KnownWordProbabilityModel : WordProbabilityModel

Statistical model used for known words in the TnT tagger. It gives the probability that a word will occur given a part of speech tag and capitalization. It stores these probabilities in a Map from WordPosCap objects to their double probabilities between 0.0 and 1.0.

Stores candidates PartOfSpeech values for words for speed, even though this data is recoverable from the probabilities map.
