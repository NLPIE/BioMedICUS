TransitionProbabilityModel

@FunctionalInterface interface TransitionProbabilityModel<S : Any, R : Any>

Returns the probability that some reduction of previous hidden states will transition to a given hidden state.

Parameters

- The type of the hidden states.

- A reduction type, some number of previous states reduced to a single object. Examples would be the single state type for first order models and a of the State types for second order models.
