candidateOf

static fun <S : Any> candidateOf(candidate: S, emissionLogProbability: Double): CandidateProbability<S>

Creates a new CandidateProbability of a candidate-state and an emission probable for that candidate state. This method would be used when a EmissionProbabilityModel needs to a provide a CandidateProbability to the viterbi processor.

Parameters

candidate - the hidden state candidate

emissionLogProbability - the base-10 log probability that the candidate state will emit an output value

- the HMM hidden state type

Return
newly created Candidate object that stores the parameters.
