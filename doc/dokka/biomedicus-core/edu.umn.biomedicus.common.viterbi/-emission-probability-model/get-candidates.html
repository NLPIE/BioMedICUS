abstract fun getCandidates(emittedValue: Y): MutableCollection<CandidateProbability<S>>

Gets the CandidateProbability objects for the emitted value. CandidateProbabilities are a hidden state along with the emission probability that this hidden state leads to the emitted value.

Parameters

emittedValue - the emitted value from the series for which hidden states are being found.

Return
a unordered collection of candidate-state emission-probability pairs.
