call

fun <T : Any> call(callable: Callable<T>): T

Runs the callable inside of a context, providing objects from that context for injected types within the context.

Parameters

callable - the Callable to run.

- the return type of the callable.

Exceptions

Exception - any exception thrown by the callable.

Return
the value of T returned by the callable.
