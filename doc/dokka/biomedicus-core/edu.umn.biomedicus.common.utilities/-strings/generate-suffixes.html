generateSuffixes

static fun generateSuffixes(string: String, maxSuffixLength: Int): Stream<String>

Creates a stream which generates all suffixes of a word up to a maximum suffix length in order of decreasing size including a final empty string. The suffixes of "computer" of max length 4 would be "uter" "ter" "er" "r" "" in that order.

Parameters

string - string to break apart into suffixes.

maxSuffixLength - the number of characters from the end that should be broken into suffixes.

Return
a stream of the suffixes of the string.
