Metadata

interface Metadata

Metadata associated with an Artifact. All Document objects on the artifact share the same metadata.

Properties

artifactID

 abstract val artifactID: String

Returns the unique artifact identifier. Generally sourced from the relative path to the source file, a primary key/integer unique identifier, or an UUID. It should be a valid relative file path in order for writers to properly function.

metadata

 abstract val metadata: MutableMap<String, String>

Returns a map of all the metadata for an artifact.

Inheritors

Artifact

 interface Artifact : Metadata

A single processing artifact, containing a number of documents of text and their associated labels.

Document

 interface Document : TextRange, Metadata

A text document and its associated label indices. Implements TextRange as the range of the document. Implements Metadata with the artifact metadata.
